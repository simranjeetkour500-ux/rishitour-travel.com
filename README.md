# Rishi Tour & Travel

Yeh repository ek simple tour & travel landing page ke liye hai.

## Files

- `index.html` - Homepage layout
- `styles.css` - Styling
- `script.js` - Basic form behavior

## Kaise chalaye

1. Browser mein `index.html` open karein.
2. Ya simple local server chalayein:
   - `python3 -m http.server 8000`
   - phir browser mein `http://localhost:8000`
